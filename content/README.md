# Visualizations Folder

This folder contains the 7 interactive HTML visualizations for the Space Debris Analysis project.

## Required Files

Upload these files here after generating them from the notebook:

1. `growth_timeline.html` - Historical accumulation of space objects
2. `regime_dominance.html` - Ownership by orbit regime
3. `launches_decays_timeline.html` - Launch/decay balance over time
4. `orbit_3d_visualization_fixed.html` - 3D spatial distribution (featured visualization)
5. `public_private_stacked_bar.html` - Public vs private sector analysis
6. `orbit_altitude_pyramid.html` - Population structure by altitude
7. `perigee_apogee_eccentricity.html` - Orbital shape diversity

## How to Generate

1. Open `GauravPatel_801426641_VA_Project1_Space_Debris (1).ipynb` in Google Colab
2. Upload `satcat.csv` data file
3. Run all cells
4. Files will be automatically saved to `/content/` in Colab
5. Download all files and upload them here

## File Size

Each HTML file is self-contained and includes Plotly.js via CDN.

Average size: 2-5 MB per file
